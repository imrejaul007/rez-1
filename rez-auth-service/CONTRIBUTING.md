# Contributing to REZ Auth Service

## Setup

```bash
git clone https://github.com/imrejaul007/rez-auth-service.git
cd rez-auth-service
npm install
cp .env.example .env
npm run dev
```

## Code Style

- Run `npm run lint` before committing
- Use TypeScript for all new files

## Testing

```bash
npm test
```

## Submitting Changes

1. Create a feature branch
2. Make your changes
3. Push and open a PR on GitHub
