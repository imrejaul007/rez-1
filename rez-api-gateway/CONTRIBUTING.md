# Contributing to REZ API Gateway

## Setup

```bash
git clone https://github.com/imrejaul007/rez-api-gateway.git
cd rez-api-gateway
npm install
cp .env.example .env
npm run dev
```

## Code Style

- Run `npm run lint` before committing
- Use TypeScript for all new files

## Submitting Changes

1. Create a feature branch
2. Make your changes
3. Push and open a PR on GitHub
